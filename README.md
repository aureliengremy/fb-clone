# <p align="center">FB Clone</p>
  
This is a project named "fb-clone" with the goal of creating a clone of the Facebook social network. The project utilizes Laravel Breeze in Vue 3, along with Tailwind CSS and Pinia.

## Usage Instructions

### Installing Dependencies
```bash
composer install
npm install
```  
### Development Script
```bash
php astisan serve
npm run dev
```
This script runs the development server, allowing users to view and test the application while it's under development.
